NLP FOR INCLUSIVE DISCOURSE: DETECTING AND MITIGATING HATE SPEECH

I developed a Python-based system to classify text as hate speech or non-hate speech using machine learning techniques. The project involved preprocessing text (tokenization, stop word removal, stemming), extracting features with TF-IDF, and experimenting with classifiers like logistic regression and neural networks. The system accepts real-time user input and demonstrates high accuracy in identifying harmful language, contributing to safer online platforms. Future work includes enhancing generalizability across datasets and languages.

❮ ❯

Project Details

  • Utilizes Python and machine learning for text classification.
  • Implements text preprocessing steps, including tokenization, stop word removal, and stemming.
  • Employs TF-IDF for feature extraction to convert text into numerical representations.
  • Experiments with classifiers like logistic regression, support vector machines, and neural networks.
  • Features a console-based interface for real-time hate speech detection.
  • Achieves high accuracy in identifying harmful language.
  • Contributes to safer digital spaces through automated content moderation.

Smart Summarizer: AI-Enhanced Multi-Language Summaries for YouTube Videos

The Smart Summarizer application uses cutting-edge AI models to generate structured, multi-language summaries for YouTube videos. It efficiently retrieves or transcribes video content using Groq's Whisper for transcription and Llama 3.1 for summarization, supporting over 12 languages. Key features include: Transcript Retrieval: Extracts YouTube transcripts or transcribes audio using Whisper with FFmpeg for audio processing. Advanced Summarization: Employs Langchain for text chunking to organize summaries into sections like Title, Overview, Key Points, and Context. Streamlit Interface: Intuitive UI for real-time input, language selection, and progress tracking. Technical Robustness: Incorporates yt-dlp for video processing, authentication cookies for private videos, and robust error handling.

❮ ❯

Project Details

  • Leverages Groq's Whisper for high-accuracy video transcription and FFmpeg for audio processing.
  • Uses Llama 3.1 for nuanced, multi-language summaries supporting over 12 languages.
  • Implements Langchain for advanced text chunking, organizing summaries into structured sections.
  • Features an intuitive Streamlit interface for real-time YouTube URL input and summary generation.
  • Integrates yt-dlp for reliable video processing and authentication cookies for private video access.
  • Includes robust error handling for invalid URLs, transcript issues, and API errors.
  • Automates temporary file cleanup to optimize system resource usage.

Bidirectional Load Balancing Strategy in Kubernetes Cluster

This project implements a bidirectional load balancing strategy in a Kubernetes cluster, optimizing traffic distribution across microservices. Leveraging Kubernetes' Ingress controllers and service discovery, the system ensures high availability and scalability. A custom Python script facilitates round-robin load balancing and enables seamless bidirectional communication between clients and services, improving system reliability and fault tolerance.

❮ ❯

Project Details

  • Leverages Kubernetes' Ingress controllers and service discovery for dynamic traffic management.
  • Implements a custom Python script to enforce a perfect round-robin load balancing pattern.
  • Optimizes resource usage to prevent pod exhaustion and enhance fault tolerance.
  • Supports bidirectional communication for seamless data flow between clients and services.
  • Enhances scalability and resilience for high-traffic microservices-based applications.
  • Demonstrates Kubernetes' effectiveness in managing distributed systems in production environments.
